The Morrison Framework™ defines identity, intelligence, safety, consciousness, qualia, and governance as geometric invariants of a controlled dynamical system. This paper demonstrates that all invariants depend on a single structural object — the time-indexed reachable manifold `ℛ(t) = Reach⁺(x₀, t)` — and that the entire stack is generated by projection operators acting on this single object. ### 3.1 Why Not a Sum

An earlier formulation expressed the master functional as:

```
Φ(ℛ) = τ(ℛ) + ∂_t μ(ℛ) + Λ ‖∇G(ℛ)‖
```

This has a dimensional mismatch: `τ(ℛ)` is a topological space, `∂_t μ(ℛ)` is a scalar, and `Λ ‖∇G(ℛ)‖` is a tensor-weighted scalar. These are fundamentally different mathematical objects and cannot be added directly.

In physics, master objects avoid this problem. The partition function in thermodynamics is a single scalar from which all observables are derived by differentiation. The metric tensor in general relativity is a single tensor from which curvature, geodesics, and energy-momentum follow. In both cases, the generating object has one consistent type, and projections extract the invariants.

The correct structure for the Morrison Framework™ follows the same principle.
